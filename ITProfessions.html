Example of JavaScript Code

To demonstrate what I have learned so far in my IT education, here is a very simple JavaScript program that will print all of the even numbers appearing in a range of 1-100.


<!DOCTYPE html>

<html>

<head>

<meta charset="UTF-8">

<title>Even Numbers</title>

<script>
function showEvenNumbers() { var display = document.getElementById('display'); var displayHTML = ""; for (var i = 1; i < 101; i++) { if (i % 2 == 0) displayHTML += "<p>" + i + "</p>"; } display.innerHTML = displayHTML }
</script>

</head>

<body onload="showEvenNumbers()">

<div id="display">

</div>

</body>

</html>
